Top Level Site Performance Tool

The purpose of this tool is to provide a useful consolidated performance report using some common performance pitfalls. We use Google Page Speed API to check our pages and display it in a pretty report for convienance. Note: this is a starting point only and serious JS performance issues in particular will need Speed Tracer or a similar tool to understand expensive methods/events, etc.
